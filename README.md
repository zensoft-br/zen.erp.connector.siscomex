## Consultar DUIMP's

- Acesse https://portalunico.siscomex.gov.br/duimp/#/consultar-duimp
- Selecione a opção **Outros parâmetros**
  - CPF do responsável: `062.607.857-12`
  - Situação: `Pré-registro`
  - Data inicial: `(informar)`
  - Clique em **Consultar**
- Anote o número da DUIMP
