Observações

  • Este serviço é uma cortesia do Zen ERP
  • Nós não armazenamos dados do certificado ou da DUIMP, a requisição é enviada diretamente para a API do Siscomex.
  • A API do Siscomex permite apenas uma autenticação por minuto, e como não armazenamos dados, é importante aguardar esse intervalo entre as requisições.
  • Documentação da API do Siscomex: https://api-docs.portalunico.siscomex.gov.br/api/plat
